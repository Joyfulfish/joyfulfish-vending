# 🐟 魚樂匯 水族販賣機網站

整合在庫報價與特價商品的全新購物平台

## ✨ 特色功能

- 🎨 **日式淺色風格**：清爽優雅的靛藍色系介面
- 📊 **階梯定價系統**：支援 1-3 階段的組合優惠
- 🛒 **購物車功能**：可一次選購多項商品統一詢價
- 🔄 **智能價格顯示**：單價商品顯示單價，組合商品顯示階梯選擇器
- 📱 **一鍵 LINE 訂購**：直接跳轉 LINE 官方帳號詢價
- 🖼️ **自動圖片匹配**：從圖片資料夾自動尋找商品圖片
- 📦 **售完展示**：保留售完商品的展示照片
- 📊 **訪客統計**：顯示在線人數和總瀏覽次數
